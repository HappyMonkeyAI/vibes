Vibes TUI High-performance terminal UI for agentic coding workflows

Vibes is an autonomous agent harness that runs in your terminal. It connects to local LLMs (Ollama, LM Studio) to plan and execute complex coding missions — breaking them into milestones, scheduling tasks with dependency resolution, and streaming results back in a rich TUI built with Ink & React.

🚀 Installation

Install via curl, npm, or clone the repo — Linux, macOS, and Windows.

⚙️ Configuration

Environment variables, .vibes/config.json, settings UI, and model setup.

📖 Usage

Creating missions, keyboard shortcuts, views, YOLO mode, and undo.

🏗️ Architecture

Agent loop, DAG scheduler, MCP, LTM, and code structure.

Quick Start

# Install globally
npm install --global vibes-tui

# Launch from any directory
vibes

Or try it without installing:

npx vibes-tui

Feature Highlights

FeatureDescription
Hierarchical mission planningBreak down high-level goals into milestones and tasks
DAG-based schedulerParallel task execution with dependency resolution
Multi-model supportSeparate models for planning, execution, review, and triage
MCP protocolConnect to external tools via stdio or SSE
Long-term memoryPersistent JSONL memory with optional Mem0 integration
Codex RAGNeo4j knowledge graph augmentation for small models
Safety netAuto-git snapshots (with scratch-workspace init), undo, thrash detection, YOLO mode
Configurable workspaceSet VIBES_LAUNCH_DIR in Settings to target any project directory
Deadlock detectionScheduler diagnoses circular task dependencies instead of hanging
Multiline missionsPaste-friendly prompt with Alt+Enter newline (SSH/Termius safe)
