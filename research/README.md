# Research

External references, comparisons, and exploratory notes for Vibes development.

## Structure

- `LINKS.md` — curated links for quick reference
- `templates/project-note.md` — template for documenting external projects
- `github-projects/` — one file per external repo/project studied
- `notes/` — deeper comparisons or design explorations

## Rules

- Record URL, license, stack, why it matters, what to cherry-pick or avoid.
- Do not copy blindly. Capture only durable, useful observations.
- If research leads to a design decision, create an ADR in `.antigravity/memories/architectural_decisions/`.
